# 德国留学签证清单 - 产品文档

## 1. 项目概述

**项目名称**: 德国留学签证清单
**项目类型**: 桌面应用（Tauri + React）
**核心功能**: 根据用户角色（学历背景）生成对应的德国留学签证材料清单，支持标记完成状态和搜索

---

## 2. 用户角色

| 角色ID | 角色名称 | 说明 |
|--------|----------|------|
| high_school | 高中毕业 | 高中毕业，去德国读本科 |
| bachelor_in_progress | 大学在读 | 本科在读，申请德国硕士 |
| bachelor_graduated | 大学毕业 | 本科毕业，申请德国硕士 |
| master_in_progress | 硕士在读 | 硕士在读，申请德国博士或二硕 |
| master_graduated | 硕士毕业 | 硕士毕业，申请德国博士或二硕 |

---

## 3. 功能需求

### 3.1 角色选择（首页）
- 展示5种用户角色供选择
- 每种角色显示对应的学历背景说明
- 选择角色后跳转到对应清单页面

### 3.2 清单展示
- 根据选择的角色显示对应的材料清单
- 材料按PDF中的顺序排列
- **原件**材料需重点标识（标记 ★）
- 支持展开/收起每项材料的详细说明
- 支持勾选标记材料准备状态
- 显示完成进度百分比

### 3.3 搜索功能
- 顶部搜索栏，快速过滤材料
- 搜索关键词高亮显示

### 3.4 数据持久化
- 本地JSON文件存储用户进度
- 关闭应用后保留勾选状态

## 5. UI/UX 设计方向

### 5.1 设计风格
- **现代前卫** - 非对称布局，独特视觉
- **亲切友好** - 柔和配色，流畅动画

### 5.2 配色方案
- 主色调：柔和渐变
- 强调色：暖橙 #F97316
- 背景：浅灰白 #F8FAFC
- 文字：深灰 #1E293B

### 5.3 字体
- 标题：Satoshi 或 Cabinet Grotesk
- 正文：Geist 或 system-ui

### 5.4 布局
- DESIGN_VARIANCE: 8 (非对称艺术布局)
- MOTION_INTENSITY: 6 (流畅动画 + 微交互)
- VISUAL_DENSITY: 4 (舒适呼吸感)

### 5.5 关键UI组件
- **角色选择页**: 垂直卡片，非对称网格，悬停磁性效果
- **清单页**: 搜索栏 + 可展开材料卡片 + 进度条
- **原件标识**: 带 ★ 标记的卡片边框高亮

---

## 6. 技术栈

- **框架**: Tauri v2 + React
- **状态管理**: React Hooks / Zustand
- **样式**: Tailwind CSS + Framer Motion
- **图标**: Phosphor Icons
- **存储**: 本地 JSON 文件
- **包管理**: npm
